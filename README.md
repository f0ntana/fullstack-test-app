# Full Stack Test

## 3. Challenge (APP)

---

#### This app has been developed with React Native

To run this project please clone

```bash
git@github.com:f0ntana/fullstack-test-app.git
```

Enter in the folder

```bash
cd fullstack-test-app
```

Now, run npm / yarn to install the dependencies

```bash
yarn
```

To start emulate

```bash
react-native run-ios
```
---
#### Third-party
react-native-render-html: `used for render the detail of article,because use html tags.`

react-native-vector-icons: `used because provider icons.`

react-navigation: `used to make routes and control of screens`
